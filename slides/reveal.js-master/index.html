Rationale

  • Provide collections with higher order functions
  • Sequential or parallel operations
  • Anonymous classes have bulky syntax

// given the interface
interface Consumer<T> { void accept(T t); }


// Java 7
pointList.forEach(new Consumer<Point>() {
    public void accept(Point p) {
        p.move(p.y, p.x);
    }
});

// Java 8
pointList.forEach(p -> p.move(p.y, p.x));

Lambda Expressions?

A function that computes an output for some combination of input values

  • (parameters) -> expression
  • (parameters) -> { statements; }

(int x, int y) -> x + y
(x, y) -> x + y
() -> 42
(String s) -> System.out.println(s)
c -> { int s = c.size(); c.clear(); return s; }

Functional Interfaces

  • Type where a lambda can be used as the argument value
  • public void forEach(Consumer consumer);
  • Interface with one abstract method
  • Sometimes called SAM - Single Abstract Method

Lambda Typing

  • Type of lambdas depends on context
  • What is the type of: x -> 2 * x ?
  • Target typing

ActionListener l = (ActionEvent e) -> ui.dazzle(e.getModifiers());

interface IntOperation { int operate(int i); }
IntOperation iop = x -> x + 2
interface DoubleOperation { int operate(int i); }
DoubleOperation dop = x -> x + 2
